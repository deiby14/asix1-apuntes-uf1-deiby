Apartado HTML

Significado=HHypertext MMarkup LLanguage

Todo comando debe tener los simbolos de apaertura <>

  1. Contenidoeste es el contenido del elemento, que en este caso es sólo texto.
  2. Elementola etiqueta de apertura, más la etiqueta de cierre, más el contenido equivale al eleme
  3. Atributo seguido por un signo de igual (=) y comillas de apertura y de cierre, encerrando el valor del atributo

Puedes también colocar elementos dentro de otros elementos. Esto se llama anidamiento.

Algunos elementos no poseen contenido, y son llamados elementos vacíos. el elemento img: img src="images/firefox-icon.png" alt="Mi imagen de prueba"> posee dos atributos, pero no hay etiqueta de cierre img> ni contenido encerrado.

  2. head/headEste elemento actúa como un contenedor de todo aquello que quieres incluir en la página HTML que no es contenido visible para otros.
  3. link/linkEn este elemento podemos añadir el comando rel para poder añadir un favicoin
  4. title/titleEn este apartado podemos retocar todo lo que tenga que ver con la parte del encabezado de nuestro documento
  5. body/bodyEn este apartado escribiremos,anñadiremos tablas,numeros,imagenes,videos ya que es la parte que afecta al cuerpo del documento

Evolución y Función

En los inicios de la web, HTML era simple y no podía representar recursos gráficos. Con el crecimiento de los sitios web, se introdujo CSS para separar estructura y estilo.

Ventajas e Inconvenientes

Las ventajas de CSS incluyen la posibilidad de mantener el código, un diseño más potente y simplicidad. Sin embargo, la inconsistencia entre navegadores puede ser un inconveniente.

Ubicación de Estilos

Los estilos se pueden ubicar en línea, internamente en la cabecera HTML, o en un documento externo. La ubicación afecta la reutilización y consistencia del diseño.

Sintaxis Básica de CSS

Una hoja de estilos tiene reglas con selectores y declaraciones. Cada declaración tiene una propiedad y su valor. Ejemplo: p { font-size: 10pt; background-color: gray; }

Composición: Margenes, Bordes y Relleno en CSS

Los márgenes, bordes y rellenos son esenciales para controlar el espacio y la apariencia de los elementos en una página web.

Google Fonts

Para integrar fuentes de Google, se puede agregar el enlace en la sección <head> y luego aplicar el estilo en los elementos deseados.

Sintaxis Básica de CSS en HTML

La sintaxis genérica de una regla CSS es la siguiente:

        
            /* Sintaxis genérica de una regla CSS */
            selector {
                propiedad_1: valor_1;
                propiedad_2: valor_2;
                /* ... más propiedades y valores ... */
            }

Por ejemplo, si queremos que todos los párrafos tengan un tamaño de letra de 10pt y un fondo de color gris, debemos definir la siguiente regla:

        
            /* Ejemplo de regla CSS para párrafos */
            p {
                font-size: 10pt;
                background-color: gray;
            }

En esta regla, 'p' es el selector con dos declaraciones: 'font-size: 10pt;' y 'background-color: gray;'. En cada declaración, la propiedad y el valor están definidos.
